# Getting started

This section takes you from a fresh clone to a running application:

1. **[Requirements](requirements.md)** — the handful of things you need installed
   (everything else is fetched for you).
2. **[Building](building.md)** — configuring and building the engine, the cooker,
   and the sample, plus the validation build.
3. **[Running the sample](running-the-sample.md)** — running `hello-triangle`,
   both windowed and as a headless single-frame capture.
4. **[Your first application](your-first-app.md)** — the anatomy of a veng app:
   the game module, the launcher, the lifecycle hooks, and the build wiring.

If you just want to read about how the engine is put together, jump to
[Concepts](../concepts/index.md) instead.
